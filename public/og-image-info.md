# Open Graph Image Requirements

To complete the SEO optimization, you need to create an Open Graph image for social media sharing.

## Image Specifications:
- **Dimensions**: 1200 x 630 pixels (recommended)
- **Format**: JPG or PNG
- **File size**: Under 1MB
- **File name**: og-image.jpg (or og-image.png)

## Content to include in the image:
- **Title**: "Image to Text Converter Online Free"
- **Subtitle**: "OCR Tool | image2word.com"
- **Visual elements**: 
  - OCR/Text recognition icon
  - Sample image with extracted text
  - Clean, professional design
  - Brand colors (blue theme)

## Where to place the image:
- Upload the image to the `public/` directory
- The image will be accessible at: `https://image2word.com/og-image.jpg`

## Tools to create the image:
- Canva (free templates available)
- Figma
- Adobe Photoshop
- Online tools like Bannerbear or Cloudinary

## Current metadata references:
The layout.tsx file already includes references to this image in:
- OpenGraph metadata
- Twitter Card metadata

Once you create and upload this image, the social media sharing will display properly.
